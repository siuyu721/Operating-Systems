An Operating System manages device communication via their respective drivers. Decides which process gets the device when and for how much time. Allocates the device in the efficient way.
Keeps track of information, location, uses, status etc. The collective facilities are often known as file system. Decides who gets the resources. Allocates the resources.
Keeps tracks of primary memory, i.e., what part of it are in use by whom, what part are not in use. In multiprogramming, the OS decides which process will get memory when and how much. Allocates the memory when a process requests it to do so.
